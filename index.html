Paris Metro

Winter 2017

University of Ottawa, Ontario

This was the final assignment of my Data Structures and Algorithms course. A .cvs file was provided, which contained a list of train stations in the paris subway system, the possible links between different stations and the time each of those links took to complete. Using this data, I built a graph and using graph traversal and search algorithms, I could determine the shortest path from one station to another using as input the starting and ending station identification numbers. The link to the github repository for this assignment is: https://github.com/wdond086/Assignments
